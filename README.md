# Cat Editor PRO 🐾

Продвинутый ASCII-редактор и симулятор котиков на Python и Tkinter с поддержкой динамических плагинов.

## ✨ Особенности проекта
- **Бессмертные котики**: По правилам проекта, коты не могут погибнуть.
- **Модульная система модов**: Безопасная загрузка плагинов из папки `./mods` с помощью встроенного контроллера разрешений (Permission Controller).
- **Кастомный ASCII-движок**: Динамический рендеринг окружения, мебели и оружия в реальном времени без съезда строк.
- **Режим Тамагочи**: Полноценный жизненный цикл, экономика монет, прокачка навыков охоты, магазин товаров и система живых автосохранений.

## 📁 Структура репозитория
- `cat_start.pyw` — Точка запуска приложения (без вызова консоли).
- `cat_gui.py` — Главный интерфейс, менеджер экранов и анимации.
- `cat_storage.py` — Логика хранения данных, экспорт `.cat` моделей и загрузчик плагинов.
- `mods/standart/tamagotchi_life.py` — Официальный масштабный плагин симулятора жизни.

## 🚀 Запуск
Убедитесь, что у вас установлен Python 3.x. Склонируйте репозиторий и запустите стартовый скрипт:
```bash
python cat_start.pyw
```
